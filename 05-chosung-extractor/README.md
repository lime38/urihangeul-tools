# 초성 추출기 (Chosung Extractor)

한글 텍스트의 각 음절에서 **초성만 추출**하는 무료 도구. '우리한글' → 'ㅇㄹㅎㄱ' 형태로 변환.

## 웹에서 바로 사용

➜ **[우리한글 — 초성 추출기](https://urihangeul.com/chosung-extractor/)**

## 주요 기능

- **초성 추출**: 한글 음절(가–힣) → 초성 19종(ㄱ/ㄲ/ㄴ/.../ㅎ)
- **공백 유지** (옵션): 공백·줄바꿈 보존 여부
- **숫자·영문 유지** (옵션): 한글 외 문자 보존 여부
- **복사 버튼**: 결과를 클립보드로 한 번에

## 활용

- 초성 퀴즈 제작 (ㅇㄹㅎㄱ → 우리한글)
- 연락처 초성 색인
- 초성 검색 기능 개발 테스트
- 한글 데이터 전처리

## 동작 원리

한글 음절은 유니코드 U+AC00~U+D7A3 범위에서 `(코드 − 0xAC00) ÷ 28 ÷ 21`로 초성 인덱스를 구합니다. 외부 라이브러리 없이 순수 JavaScript로 구현.

## 로컬에서 실행

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없는 단일 페이지.
