# 한영 타자 변환기 (Keyboard Layout Converter)

한글/영문 입력 모드를 **착각하고 잘못 입력한 텍스트**를 자동으로 변환하는 무료 도구. `dnflgksrmf` → `우리한글` 같은 두벌식 자판 기준 양방향 변환.

## 웹에서 바로 사용

➜ **[우리한글 — 한영 타자 변환기](https://urihangeul.com/keyboard-layout-converter/)**

## 주요 기능

- **양방향 변환** — 영→한 / 한→영 토글
- **두벌식 자판 기준**
- **쌍자음** (ㄲ, ㄸ, ㅃ, ㅆ, ㅉ) 지원
- **이중 모음** (ㅘ, ㅙ, ㅚ, ㅝ, ㅞ, ㅟ, ㅢ) 지원
- **겹받침** (ㄺ, ㄻ, ㄼ 등) 정확 변환
- **복사 버튼**

## 활용

- 한/영 전환 깜빡한 메신저·메일·문서 텍스트 즉시 복구
- 'dnflgksrmf' → '우리한글', '안녕' → '소ㅜs' 같은 입력 사고 정리
- 한글 데이터셋 정제 (잘못 입력된 키 분리)

## 동작 원리

**영→한:** 영문 키 → 한글 자모 매핑 (EN2KO) → 자모 시퀀스 → 한글 음절 조합 (composeKorean). 초성·중성·종성 + 이중 모음 (DBLVOWEL) + 겹받침 (DBLCONG) 처리.

**한→영:** 한글 음절 분해 (decompose) → 자모별 영문 키 매핑 (KO2EN). 이중 모음·겹받침은 영문 두 키 시퀀스로 (CPVOWEL / CPCONG).

외부 라이브러리 없이 순수 JavaScript로 구현.

## 한계

- 두벌식 한정 (세벌식·콜맥 미지원)
- 일부 이중 자모 분해 시 영문 시퀀스가 정확하지 않을 수 있음 (자판마다 변형)

## 로컬에서 실행

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없는 단일 페이지.
