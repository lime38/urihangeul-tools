# 공백 정리기 (Whitespace Cleaner)

웹페이지·PDF·문서에서 복사한 텍스트의 **연속 공백·빈 줄·앞뒤 공백**을 한 번에 정리하는 무료 도구. 옵션 5개로 정리 강도 조절.

## 웹에서 바로 사용

➜ **[우리한글 — 공백 정리기](https://urihangeul.com/whitespace-cleaner/)**

## 주요 기능

### 기본 정리 (체크 시 적용)
- **각 줄 앞뒤 공백 제거**
- **연속 공백 하나로** (탭 포함)
- **연속 빈 줄 하나로** (단락 구분 유지)

### 추가 옵션
- **모든 줄바꿈 제거**: 전체를 한 줄로 합치기 (스크래핑 정리에 유용)
- **빈 줄 모두 제거**: 본문 사이 모든 빈 줄 삭제

## 활용

- PDF·웹페이지에서 복사한 텍스트 정리
- 스크래핑 데이터 전처리
- 긴 글을 한 줄로 합치기 (RAG 입력용 등)
- 깔끔한 문서 작성

## 로컬에서 실행

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없는 단일 페이지.
