# 글자 수 세기 (Word Counter)

텍스트의 **글자 수(공백 포함/제외)**, **어절 수**, **문장 수**, **단락 수**를 입력 즉시 실시간으로 보여 주는 무료 도구.

## 웹에서 바로 사용

➜ **[우리한글 — 글자 수 세기](https://urihangeul.com/word-counter/)**

## 주요 기능

- 글자 수 (공백 포함/제외) 동시 표시
- 어절 수 (공백 구분 단어 단위)
- 문장 수 (마침표·느낌표·물음표·줄바꿈 기준)
- 단락 수 (빈 줄 구분)
- 입력 즉시 실시간 업데이트

## 활용

- 자기소개서 / 보고서 글자 수 맞추기
- SNS·블로그 글 분량 확인
- 원고 어절·문장 수 점검

## 로컬에서 실행

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없는 단일 페이지.
