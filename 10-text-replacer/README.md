# 찾아 바꾸기 (Text Replacer)

특정 단어/문구를 찾아 **일괄 치환**하는 무료 도구. 정규식과 대소문자 구분 옵션 지원.

## 웹에서 바로 사용

➜ **[우리한글 — 찾아 바꾸기](https://urihangeul.com/text-replacer/)**

## 주요 기능

- **단순 치환** — 찾을 텍스트 → 바꿀 텍스트
- **삭제** — 바꿀 텍스트 비우면 일괄 삭제
- **정규식 사용** — 패턴 매칭 (예: `\d+` 숫자 전체)
- **대소문자 구분** — 끄면 `Hello`/`hello` 같이 처리
- **치환 건수 표시** — "N건 치환되었습니다"
- **복사 버튼** — 결과 클립보드 즉시 복사

## 활용

- 오탈자 일괄 수정 (이전 표기 → 새 표기)
- 데이터 노이즈 제거 (URL/이모지/특정 패턴)
- 마크다운/HTML 태그 일괄 변경
- 이름·용어 통일 (영문 표기 → 한글 등)

## 동작 원리

`new RegExp(find, flags)`로 동적 패턴 생성. 정규식 비활성 시 메타문자(`.*+?^${}()|[]\`) 이스케이프 처리. 플래그는 항상 `g`(global) + 대소문자 옵션 시 `i` 추가.

`String.prototype.match()`로 매칭 건수 카운트 → `String.prototype.replace()`로 치환. 정규식 파싱 실패 시 오류 메시지 표시.

외부 라이브러리 없이 순수 JavaScript로 구현.

## 한계

- 매우 큰 입력(수십 MB 이상)은 브라우저 메모리 한계
- 정규식 백레퍼런스(`$1`/`$2` 등)는 표준 `String.replace` 동작에 의존 — 복잡한 캡처 그룹 치환은 정규식 모드 필수
- Undo 없음 — 치환 전 원본을 직접 보관할 것

## 로컬에서 실행

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없는 단일 페이지.
