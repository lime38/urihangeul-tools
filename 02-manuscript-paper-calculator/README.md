# 원고지 계산기 (Manuscript Paper Calculator)

입력한 텍스트가 **200자/400자/600자 원고지 기준으로 몇 매**인지 자동 계산하는 무료 도구. 독후감/논술/공모전 원고 분량 확인에 씁니다.

## 웹에서 바로 사용

➜ **[우리한글 — 원고지 계산기](https://urihangeul.com/manuscript-paper-calculator/)**

## 주요 기능

- 200자/400자/600자 원고지 기준 선택
- 매 수 + 남은 칸까지 정확히 표시
- 공백 제외 글자 수 동시 표시
- 입력 즉시 실시간 계산

## 활용

- 독후감/논술 과제 분량 확인
- 공모전 원고 매수 점검
- 글짓기 대회 분량 맞추기

## 로컬에서 실행

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없는 단일 페이지.
