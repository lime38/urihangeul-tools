# 줄 정렬기 (Line Sorter)

한 줄에 하나씩 입력된 텍스트 목록을 **가나다 순/역순/뒤집기**로 정렬하는 무료 도구. 중복 줄/빈 줄 제거 옵션 제공.

## 웹에서 바로 사용

➜ **[우리한글 — 줄 정렬기](https://urihangeul.com/line-sorter/)**

## 주요 기능

- **가나다 순 정렬** — 한국어 로케일(`localeCompare('ko')`) 기준
- **역순 정렬** — 가나다 역순
- **순서 뒤집기** — 입력 순서 위아래 반전
- **중복 줄 제거** — 같은 줄 한 번만 남김 (Set)
- **빈 줄 제거** — 공백만 있는 줄도 포함
- **복사 버튼** — 결과 클립보드 즉시 복사

## 활용

- 단어 목록/이름 명단 가나다 순 정리
- 데이터셋 줄 단위 dedup
- 로그/CSV 행 단위 정렬 전처리
- 책 제목/링크 모음 알파벳 순 정렬

## 동작 원리

`textarea` 입력을 `\n` 기준 분할 → 옵션(`trim_empty`/`dedup`) 적용 → `localeCompare('ko')` 정렬. 한국어·영문 혼재 시도 한국어 로케일이 합리적 순서를 잡아 줍니다.

외부 라이브러리 없이 순수 JavaScript로 구현.

## 한계

- 매우 큰 입력(수십만 줄 이상)은 브라우저 메모리 한계. 일반 목록 정리(수천 줄 이내)에 적합
- 자연어 정렬(숫자가 섞인 'item2'/'item10' 등)은 단순 사전 순으로 처리 — 자연 정렬 옵션 별도 미제공

## 로컬에서 실행

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없는 단일 페이지.
