# 자모 분리기 (Jamo Splitter)

한글 음절을 **초성/중성/종성 자모로 분리**하거나, 자모를 다시 음절로 결합하는 무료 도구. 겹받침 11종도 자동 분해.

## 웹에서 바로 사용

➜ **[우리한글 — 자모 분리기](https://urihangeul.com/jamo-splitter/)**

## 주요 기능

- 분리 모드: 음절 → 초성/중성/종성 자모
- 결합 모드: 자모 시퀀스 → 음절 (역방향)
- 겹받침 11종 자동 분해 (ㄺ→ㄹㄱ, ㄼ→ㄹㅂ 등)
- 한글 이외 문자(영문/숫자/공백)는 그대로 유지

## 활용

- NLP 개발 (한국어 토큰화 전처리)
- 언어학 연구 (음운 분석)
- 초성 퀴즈 / 자모 퍼즐 제작
- 한국어 학습 보조

## 로컬에서 실행

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없는 단일 페이지.
