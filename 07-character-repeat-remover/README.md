# 글자 반복 제거기 (Character Repeat Remover)

'ㅋㅋㅋㅋ' → 'ㅋ' 하나로, '오오오오오' → '오'처럼 과하게 반복된 같은 문자를 지정한 최대 횟수 이하로 줄이는 무료 도구. SNS 게시글, 채팅 기록, 댓글 데이터 전처리에 사용.

## 웹에서 바로 사용

➜ **[우리한글 — 글자 반복 제거기](https://urihangeul.com/character-repeat-remover/)**

## 주요 기능

- **반복 기준 조절** — 1~9 사이에서 초과분만 잘라 줍니다
- **2회 반복 유지 옵션** — 'ㅋㅋ', 'ㅠㅠ' 같이 자연스러운 두 번 반복은 살림
- **자모만 처리 옵션** — 자음·모음(ㅋ, ㅎ, ㅠ 등)만 줄이고 일반 글자는 그대로 둠
- **복사 버튼** — 결과를 클립보드로 한 번에

## 활용

- 카톡·문자 대화 캡처를 글로 옮길 때 반복 문자 정리
- SNS 댓글·게시글 데이터 분석 전처리
- 채팅·로그 데이터셋 정제
- 강조 표현(ㅠㅠㅠㅠ)을 통일된 형태(ㅠㅠ)로 표준화

## 동작 원리

문자열을 한 글자씩 훑으며 같은 글자가 연속될 때 그 길이를 셉니다. 길이가 `limit`을 넘으면 `limit`만큼 잘라서 결과에 붙입니다. `keep_double` 옵션이 켜져 있으면 `limit`을 최소 2로 유지해 두 번 반복은 보존. `jamo_only` 옵션은 유니코드 자모 영역(U+3131~U+314E 자음 / U+314F~U+3163 모음)만 정리 대상으로 삼고 나머지 문자는 그대로 둡니다. 외부 라이브러리 없이 순수 JavaScript로 구현.

## 예시

| 입력 | 옵션 | 결과 |
|---|---|---|
| `재밌었어ㅋㅋㅋㅋㅋ` | 최대 1, 2회 유지 ON | `재밌었어ㅋㅋ` |
| `어머어머어머` | 최대 1, 자모만 OFF | `어머` |
| `정말 정말 좋아요ㅠㅠㅠㅠ` | 최대 1, 자모만 ON | `정말 정말 좋아요ㅠㅠ` |
| `오오오오 대박!!!!` | 최대 2, 2회 유지 OFF | `오오 대박!!` |

## 한계

- *연속된 동일 문자*만 인식 — `ㅋㅎㅋㅎㅋㅎ` 같은 *번갈아 가며 반복*은 처리 안 함
- 한글 음절(가, 나, 다…)도 자모 옵션을 끄면 정리 대상이 됨 — 의도와 다를 경우 *자모만* 옵션을 권장
- 한 글자 단위 비교라 *단어 단위* 반복(예: '진짜 진짜 진짜')은 정리하지 않음

## 로컬에서 실행

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없는 단일 페이지.
