# 로마자 표기 변환기 (Romanization Converter)

국립국어원 **로마자 표기법(2000년 고시)** 에 따라 한국어를 영문으로 변환하는 무료 도구. '서울' → 'seoul', '홍길동' → 'honggildong' 형태.

## 웹에서 바로 사용

➜ **[우리한글 — 로마자 표기 변환기](https://urihangeul.com/romanization-converter/)**

## 주요 기능

- **국립국어원 표기법(2000) 기준 변환**
- **초성/중성/종성 로마자 값 각각 적용** — 음절 단위 직접 변환
- **한글 이외 문자(영문/숫자/특수문자) 그대로 유지**
- **복사 버튼**: 결과를 클립보드로 한 번에

## 활용

- 여권 영문 이름 표기 참고
- 명함·이메일 서명 영문 표기
- 주소 영문 변환 (지번·도로명)
- 콘텐츠 영문 슬러그·식별자 생성

## 동작 원리

한글 음절은 유니코드 U+AC00~U+D7A3 범위에서 `(코드 − 0xAC00) ÷ 588`로 초성, `÷ 28 mod 21`로 중성, `mod 28`로 종성 인덱스를 구합니다. 각 인덱스에 해당하는 로마자 값(`RC`/`RV`/`RJ` 배열)을 이어 붙여 변환. 외부 라이브러리 없이 순수 JavaScript로 구현.

## 한계 (음운 규칙)

음절 단위 직접 변환 방식이라 다음 음운 규칙은 *단순화*됐을 수 있습니다.

- **연음** (받침이 다음 음절로 옮아 발음) — 예: '독립' [동닙]
- **구개음화** — 예: '같이' [가치]
- **자음 동화** — 예: '신라' [실라]

여권·공식 문서 표기는 결과를 *참고*로 사용하고 [국립국어원 한국어 어문 규범](https://www.korean.go.kr/) 기준으로 최종 확인하세요.

## 로컬에서 실행

`index.html` 파일을 브라우저에서 열면 됩니다. 외부 의존성 없는 단일 페이지.
